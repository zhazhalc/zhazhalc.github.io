年轻人的第一个博客

前言

你是否在碰到error的时候，搜索后发现解决办法就在某个大佬的博客里，你是否也想用一篇篇博客去记录自己的进步与解决掉的问题，那就来搭一个个人博客吧，虽然博客园、CSDN等等都可以写博客还更方便，但是有一个属于自己的博客网站感觉更有b格(坂本推眼镜.jpg)，嗯！
本文使用hexo+github pages来搭建博客，是像我这样的小白的第一选择

什么是 Hexo？
Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用>Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

关于 GitHub Pages 

GitHub Pages 是一项静态站点托管服务，它直接从 GitHub 上的仓库获取 HTML、CSS 和 JavaScript 文件，（可选）通过构建过程运行文件，然后发布网站。 可以在 GitHub Pages 示例集合中看到 GitHub Pages 站点的示例。

你可以在 GitHub 的 github.io 域或自己的自定义域上托管站点。 有关详细信息，请参阅“配置 GitHub Pages 站点的自定义域”。

注：本文所用系统是win11

主题还挺多的，选到了喜欢的就进入该主题的github仓库，在我们本地的Hexo源码目录下的themes目录git clone该主题，下完了就可以看到themes目录下多出来了一个hexo-theme-<主题名>的文件夹

更改主题配置

打开hexo根目录下的_config.yml文件，theme默认应该是landscape，就是最开始的那个主题，现在你需要把theme后面的内容改成你下的主题文件夹的名字，一定要保持一致

五、日常更新博客

我们新增用来生成静态网页的资源都是放在Hexo源码目录下的source/_posts目录下的，所以只要你把你写好的md文件放在这个目录下面，然后部署到github pages上就行。
